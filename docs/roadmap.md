# Roadmap

## v0.1

- Device export ingestion and normalization
- Local SQLite canonical store
- Notion auto-provision and upsert worker
- Manual and scheduled sync
- Core desktop screens

- Cloud notebook connector with Playwright
- Auth state machine (`connected`, `needs_auth`, `reconnected`)
- Session-expiry UI with reconnect prompt

## v0.3

- Preview-first-sync flow
- Library and passage filtering
- Configurable sync frequency

## v1.0

- Reliability hardening and broader fixture coverage
- Signed/notarized macOS release artifacts
- OSS contribution docs and extension model
